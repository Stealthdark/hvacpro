## Image Optimization Tips

### Recommended Sizes

**Hero Background:**
- Desktop: 1920x1080px
- Mobile: 1080x1920px (portrait)
- Format: JPG (optimized)
- Size: < 500KB

**Product Images:**
- Card size: 600x450px
- Detail page: 800x600px
- Format: JPG or WebP
- Size: < 150KB each

**About Us Images:**
- Desktop: 800x600px
- Mobile: 600x450px
- Format: JPG
- Size: < 200KB

**Icons & Graphics:**
- Format: SVG (preferred) or PNG
- SVG: Vector format (scalable)
- PNG: @2x resolution for retina

### Optimization Tools

**Online Compressors:**
1. **TinyPNG:** https://tinypng.com (Best for PNG/JPG)
2. **Squoosh:** https://squoosh.app (Google's image compressor)
3. **ImageOptim:** https://imageoptim.com (Mac app)
4. **Compressor.io:** https://compressor.io

**Bulk Optimization:**
- Use ImageOptim (Mac) or FileOptimizer (Windows)
- Compress all images before uploading
- Target: 70-80% quality for web

## Implementation Instructions

### Option 1: Use Direct Unsplash URLs (Current)

**Pros:**
- No file management needed
- Always up-to-date
- Automatic optimization by Unsplash

**Cons:**
- Requires internet connection
- Slight dependency on Unsplash servers

**Code Example:**
```html
<img src="https://images.unsplash.com/photo-XXXXX?w=800&h=600&fit=crop&q=80"
     alt="Description">
```

### Option 2: Download and Host Locally

**Steps:**
1. Download images from Unsplash
2. Optimize using TinyPNG or Squoosh
3. Save to `assets/images/` folder
4. Update HTML src paths

**Pros:**
- Full control over images
- No external dependencies
- Slightly faster load times

**Cons:**
- File management required
- Manual optimization needed

**Code Example:**
```html
<img src="assets/images/daikin-central.jpg"
     alt="Daikin Centralized AC">
```

## URL Parameters for Unsplash

### Useful Parameters

**Size:**
- `w=800` - Width in pixels
- `h=600` - Height in pixels

**Quality:**
- `q=80` - Quality (0-100, recommend 75-85)

**Fit:**
- `fit=crop` - Crop to exact dimensions
- `fit=clip` - Don't upscale, clip if needed
- `fit=max` - Fit within dimensions

**Format:**
- `fm=jpg` - Force JPG format
- `fm=webp` - Force WebP (modern browsers)
- `auto=format` - Auto-select best format

**Example:**
```
https://images.unsplash.com/photo-1631545835535-4a5faa7bb1ed?w=800&h=600&fit=crop&q=80&fm=webp
```

## Troubleshooting

### Images Not Loading?

**Check:**
1. Internet connection active?
2. Unsplash URL correct?
3. Browser cache cleared?
4. Check browser console for errors

**Solutions:**
- Clear browser cache (Ctrl+Shift+R / Cmd+Shift+R)
- Try incognito/private mode
- Check if Unsplash is accessible
- Use fallback onerror attribute

### Slow Loading?

**Optimize:**
1. Reduce image dimensions (w= parameter)
2. Lower quality (q=75 instead of q=100)
3. Use WebP format (fm=webp)
4. Implement lazy loading

**Lazy Loading Code:**
```html
<img loading="lazy"
     src="https://images.unsplash.com/photo-XXXXX?w=800&q=80"
     alt="Description">
```
